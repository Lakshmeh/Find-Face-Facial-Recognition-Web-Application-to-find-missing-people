Why Find Face?

This website uses advanced face recognition technology to find missing people. It is a cross-platform where lost-and-found people can contact. You can find other relevant resources and helpline numbers for the same here. Alongside are some things to keep in mind while using this web application:

Keep images of the missing person/recovered person handy

This web application is easy to use and user-friendly. All you have to do is just either show the missing person's photo through the webcam, if the web application is able to recognise the missing person correctly, the most recently updated data regarding their location and contacts can be retrieved.

Prone to some inaccuracy

This web application is at its initial stages at the moment, so the face recognition technology is prone to inaccuracy due to small size of data base it contains.

Be accurate while filling forms here

As someone who recovered a missing person, it is very important to be precise about their current location, along with accurate your accurate phone number.

Call To Action

This website puts advanced facial recognition technology to use. All you have to do is report if you lost or recovered someone. If you come accross anyone who appears lost and away from their family, please report about their details here. Sparing five minutes for something like this could save lives.
